## 📌 Project Overview

This repository is my **Digital Literacy Portfolio**, submitted as part 
of the CSE0001 course at VIT Bhopal University. For this project, I 
took on the role of a **Student Digital Ambassador** — someone 
responsible for helping peers navigate the digital world safely, 
professionally, and responsibly.

The project consists of five tasks, each mapped to one module of the 
course. Together, these tasks cover the core pillars of digital literacy: 
understanding digital tools, building a professional online presence, 
practising coding, communicating professionally, and staying safe from 
cybercrime. All work is documented in this single GitHub repository along 
with a full written project report.

---

## 📚 Module Summaries

### 📊 Module 1 – Digital Literacy Awareness Infographic
**Task:** Created a one-page infographic using Canva covering three topics —
what digital literacy is, useful digital tools for students, and safe 
internet practices. The infographic was designed to help batchmates 
quickly understand why digital literacy matters in today's world.

---

### 🧑‍💼 Module 2 – Student Digital Portfolio
**Task:** Set up professional profiles on GitHub, Stack Overflow, and 
HackerRank. GitHub was used to host code projects and a profile README. 
Stack Overflow was set up to engage with the global developer community. 
HackerRank was used for coding practice — a 4-star Python badge was 
earned. These platforms form the foundation of my long-term professional 
digital presence.

---

### 🖥️ Module 3 – Coding & Collaboration Platforms
**Task:** Completed the itertools.product() Python challenge on HackerRank 
with all test cases passing. Also created a five-question Digital Literacy 
Awareness Quiz on Google Forms — including MCQs and short answer questions 
— linked to a Google Sheet for response collection.

---

### ✉️ Module 4 – Professional Email & Etiquette Guide
**Task:** Wrote two professional emails — one requesting a deadline 
extension from a professor, and one expressing interest in a summer 
internship. Also created a Social Media Do's and Don'ts checklist with 
5 Do's and 5 Don'ts for responsible social media use as a college student.

---

### 🔐 Module 5 – Cybercrime Awareness Case Study
**Task:** Researched and wrote a 200+ word case study on UPI/Online 
Payment Fraud covering how the crime works, who is targeted, and its 
consequences. Also created a Stay Safe Online checklist with 8+ 
actionable prevention tips specific to college students in India, 
including UPI safety and official reporting channels.
